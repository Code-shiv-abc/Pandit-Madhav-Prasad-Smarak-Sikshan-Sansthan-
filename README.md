## Project Setup

This is a modern, immersive school website built with Next.js, Tailwind CSS, and Framer Motion.

### Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
